*v_repExtRomeo Plugin*

****************************************************************************************************************************************************
This plugin is designed to allow the interaction of a Romeo robot model built in V-REP with some Aldebaran softwares (NAOqi, Choregraphe) useful to create and edit movements and interactive behaviours.
****************************************************************************************************************************************************

In order to build and run the plugin:

1. Paste the folder "scripts" in your simulator-sdk-VERSION
2. Run the script "moveLib.sh" inside this folder
3. Run the script "launch.sh" in the folder "v_repExtRomeo"
4. Run V-REP and Choregraphe and set the connection between the robot models

Enjoy the plugin!
